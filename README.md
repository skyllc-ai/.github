# skyllc-ai/.github

This repository holds default community health and metadata files for the
[**Sky, LLC**](https://github.com/skyllc-ai) GitHub organization.

- [`profile/README.md`](profile/README.md) — the landing page rendered at
  [`github.com/skyllc-ai`](https://github.com/skyllc-ai).
- `profile/assets/` — images used by the profile README.

This repo intentionally contains no code. Changes here affect how the
organization appears on GitHub; they do not affect any shipped software.

For the flagship product, see
[`skyllc-ai/UltraFastFileSearch`](https://github.com/skyllc-ai/UltraFastFileSearch).
